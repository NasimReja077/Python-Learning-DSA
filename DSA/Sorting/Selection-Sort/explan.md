## Complete Code with Line Numbers

```python
# Selection Sort Program                    # Line 1 (comment)
                                            # Line 2 (blank)
def selection_sort(arr):                    # Line 3
    n = len(arr)                            # Line 4
                                            # Line 5 (blank)
    # Traverse through all array elements   # Line 6 (comment)
    for i in range(n - 1):                  # Line 7
                                            # Line 8 (blank)
        # Assume the current position holds the minimum element  # Line 9 (comment)
        min_index = i                       # Line 10
                                            # Line 11 (blank)
        # Find the minimum element in remaining unsorted array  # Line 12 (comment)
        for j in range(i + 1, n):          # Line 13
            if arr[j] < arr[min_index]:     # Line 14
                min_index = j               # Line 15
                                            # Line 16 (blank)
        # Swap the found minimum element with first element  # Line 17 (comment)
        arr[i], arr[min_index] = arr[min_index], arr[i]  # Line 18
                                            # Line 19 (blank)
    return arr                              # Line 20
                                            # Line 21 (blank)
                                            # Line 22 (blank)
# -------- Main Program --------            # Line 23 (comment)
numbers = [64, 25, 12, 22, 11]             # Line 24
                                            # Line 25 (blank)
print("Original Array:", numbers)           # Line 26
                                            # Line 27 (blank)
selection_sort(numbers)                     # Line 28
                                            # Line 29 (blank)
print("Sorted Array:", numbers)             # Line 30
```

## LINE 7: Outer Loop (Position Selection)

```python

```

**What it does:**
- Iterates through positions to fill with correct elements
- `range(n - 1)` = `range(4)` = 0, 1, 2, 3
- Need only n-1 iterations (last element auto-sorted)

**Why n - 1?**
```
5 elements need 4 passes:
i=0: Find minimum for position 0
i=1: Find minimum for position 1
i=2: Find minimum for position 2
i=3: Find minimum for position 3
i=4: (Skip - position 4 automatically has last element)
```


### **2. Tuple Unpacking Swap**
```python
# Python's elegant swap

# Equivalent to:
temp = a
```
